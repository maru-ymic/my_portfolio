    ポートフォリオサイト

    • 作品について

      木材の加工や配送を行う企業様のリニューアルサイトを作成しました。企業実習先での課題ですので、トップページのみで、会社名は架空のものに変更しています。

    • 課題

      サイトが古くなってきたので新しくしたい。環境対策商品を扱っているため、SDGsロゴを導入してほしい。

    • 目的

      木材の種類や商品紹介をより詳細にすることで、新規のお客様にも安心して利用いただけることを目的としました。

    • 制作期間

      1ヶ月
      内訳：
      情報収集：１週間
      デザイン：１週間
      コーディング：２週間

    • 制作担当

      デザイン、コーディング

    • 使用技術

      HTML , CSS , JavaScript

    • 使用ツール

      Adobe Illustrator , Adobe Photoshop , Adobe XD , Visual Studio Code

  • 駐艇場の新規サイト
    （職業訓練校 実制作）

    駐艇場の新規サイト

    • 作品について

      駐艇場として船舶の保管をされている会員制マリーナの新規WEBサイトを作成しました。
      ※ レスポンシブ非対応となっています。

    • 課題

      デジタルコンテンツでの発信がないため、予約状況など会員様からお問い合わせがある。

    • 目的

      会員様向けに琵琶湖の波の状況や風速などの天候や現在の予約状況が把握できるコンテンツを導入。デジタル化に抵抗のある方でも便利なイメージを持ってもらうことでサイト利用に繋げる。

    • 制作期間

      1ヶ月
      内訳：
      企画：１週間
      デザイン：２週間
      コーディング：１週間少々

    • 制作担当

      ５人メンバーの全体の進行管理を担当
      デザイン、コーディングを担当

    • 使用技術

      HTML , CSS , JavaScript

    • 使用ツール

      Adobe Illustrator , Adobe Photoshop , Adobe XD , Visual Studio Code

    • 作品 URL

      https://mitchael.web.fc2.com/
